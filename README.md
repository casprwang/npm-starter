# npm-starter
> starter for a fresh npm package with babel and jest

## Install
```
$ git clone https://github.com/wangsongiam/npm-starter.git <project-name>
$ cd <project-name>
```

## Entry point
### Development
```
./index.js
```
### Publish
```
./lib/index.js
```

## Testing (on watch)
```
$ npm run build
$ npm run dev
```

## Translating with Babel (on watch mode)
```
npm run build
```
